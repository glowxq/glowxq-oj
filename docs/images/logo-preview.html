🎨 设计特色

  • 配色方案: 绿色到蓝色渐变 (#10B981 → #06B6D4 → #3B82F6)
  • 萤火虫元素: 5只动态萤火虫，象征代码灵感的闪现
  • 动画效果: 发光动画、透明度变化、飞行轨迹
  • 编程主题: 代码括号和代码行，体现OJ平台特色
  • 矢量格式: SVG格式，支持任意缩放不失真
  • 现代设计: 渐变、发光效果，符合现代UI设计趋势

🐛 萤火虫寓意

  • 灵感闪现: 萤火虫的光芒象征编程时的灵感瞬间
  • 夜间学习: 代表程序员夜以继日的学习精神
  • 微光汇聚: 每个学习者都是一点微光，汇聚成知识的海洋
  • 生机活力: 绿色基调体现生机与成长
  • 智慧之光: 蓝色元素代表理性思维与智慧
