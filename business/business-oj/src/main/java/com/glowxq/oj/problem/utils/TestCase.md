=== 标准格式 ===
**用例 1**

**输入：**

```
4 2
1 3
2 4
```

**输出：**

```
2
```

---

**用例 2**

**输入：**

```
3 0
```

**输出：**

```
6
```

---

**用例 3**

**输入：**

```
3 2
1 2
2 1
```

**输出：**

```
0
```

| 用例 | 输入                  | 输出  |
|----|---------------------|-----|
| 1  | `4 2<br>1 3<br>2 4` | `2` |
| 2  | `3 0`               | `6` |
| 3  | `3 2<br>1 2<br>2 1` | `0` |
